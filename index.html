SCSS - PRESET

Le main.scss fait office de controleur pour le style.
il va notamment s'occuper d'importer le(s) resets css, les différentes librairies (ou thèmes) et enfin tout nos partials !

Les "partials" sont des fichiers préfixés d'un underscore (le fameux "tiret du 8" : _) afin de ne pas être compilés et de ne pas apparaître dans la liste des fichiers CSS générés.

le fichier partials/all.scss permet justement de charger tout nos partials (modules, composants et pages).

Les modules représente notre base pour l'écriture du style.
Ils reprennent, entre autre, toutes les variables, le style global appliqué partout (body, h1, p...), les fonts etc...

Les composants représentent des petites parties de site bien spécifique et réutilisable. Le footer, les boutons, les formulaires par exemple...

Les pages représente le style globale d'une page spécifique (hors composantes) ET/OU les pages annexes qui utilisent un style différent.
